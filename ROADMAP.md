# 🗺️ Project Roadmap

This document outlines the planned development milestones for the project.  
Each milestone represents a key stage of progress toward achieving full feature parity with Minecraft.

---

## 🚧 Version 0.x Series — Foundations and Core Systems

### **v0.0.10 — Basic Lighting & Chunk Generation**
**Goal:** Establish the foundation for a functional voxel world.  
**Key Features:**
- Implement basic chunk generation (static terrain for now)
- Add simple block lighting system (emissive and ambient light propagation)
- Ensure world rendering and chunk management are stable
- Begin groundwork for saving/loading chunks

---

### **v0.0.2 — Implement basic block types**
**Goal:** Establish all basic block types and get them rendering, ie stairs and water and leaves.  
**Key Features:**
- Implement basic chunk generation (static terrain for now)
- Add simple block lighting system (emissive and ambient light propagation)
- Ensure world rendering and chunk management are stable
- Begin groundwork for saving/loading chunks

---

## 🌍 Version 1.x Series — Feature Parity with Minecraft

### **v1.0.0 — Full Feature Parity**
**Goal:** Reach complete feature parity with vanilla Minecraft (Java Edition).  
**Key Features:**
- Full survival mechanics (crafting, mining, combat, health, hunger)
- Redstone logic system
- Biomes, structures, mobs, and entities
- Weather and day/night cycle
- Advanced lighting and shading system
- World saving/loading and multiplayer synchronization

---

## 📅 Future Milestones (Planned)
- **v1.1.x** — Modding API & Custom Content System
- **v1.2.x** — Performance optimization & rendering overhaul
- **v2.0.0** — Introduce new gameplay mechanics beyond Minecraft
